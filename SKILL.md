# 盈米基金舆情周报生成 Skill

此 Skill 自动从 Excel 素材文件生成盈米基金舆情监测周报。

## 功能

- 自动读取多个 Excel 工作表中的舆情数据
- 筛选权威媒体报道，排除转载网站和公告类内容
- 智能去重（按摘要和标题）
- 生成包含以下内容的 Word 周报：
  - 盈米基金重点信息
  - 竞品要闻
  - 合作伙伴要闻
  - 行业要闻

## 使用方法

### 快速开始

提供以下信息：
1. **主数据文件**：包含所有监测数据的 Excel 文件（多个工作表）
2. **官方媒体报道文件**（可选）：官方媒体报道链接的 Excel 文件
3. **监测时间范围**：开始日期和结束日期
4. **输出路径**：生成的 Word 文件保存位置

### 示例

```
请生成本周的舆情周报：
- 主数据文件：/path/to/20260105080000-20260112080000.xlsx
- 官方媒体报道：/path/to/官方媒体报道链接情况.xlsx
- 时间范围：2026年01月04日 - 2026年01月12日
- 输出到：/path/to/盈米基金舆情周报.docx
```

### 参数说明

| 参数 | 说明 | 必需 |
|------|------|------|
| `--data-file` | 主数据 Excel 文件路径 | 是 |
| `--official-file` | 官方媒体报道 Excel 文件路径 | 否 |
| `--output` | 输出 Word 文件路径 | 是 |
| `--start-date` | 监测开始日期 | 是 |
| `--end-date` | 监测结束日期 | 是 |

## 输入文件格式

### 主数据文件结构

Excel 文件需包含以下工作表：
- **盈米相关**：主品牌-盈米基金、关键人物、主产品-且慢等
- **竞品**：E大、基金豆、翼支付、创必承等
- **合作伙伴**：中航证券、粤开证券等
- **银行券商**：易方达、国联证券、中金财富等
- **行业监管**：监管政策法规、基金处罚违规

每个工作表至少包含以下列：
- 序号、监测专题名称、标题（支持 HYPERLINK）、发布时间、倾向性、来源网站等

### 官方媒体报道文件结构

包含以下列：
- 序号、媒体、发布时间、选题、标题、记者、署名、链接

## 配置说明

Skill 配置位于 `scripts/config.json`，包含：

- **权威媒体列表**：24家权威媒体名称
- **转载网站列表**：需要排除的转载网站
- **公告关键词**：用于识别和排除公告类内容
- **盈米关键词**：用于筛选盈米相关摘要
- **竞品/合作伙伴/行业工作表列表**：需要读取的工作表名称

## 处理逻辑

1. **数据读取**：从 Excel 文件的各个工作表读取数据
2. **筛选**：
   - 只保留权威媒体报道
   - 排除转载网站
   - 排除公告类内容
3. **去重**：
   - 按摘要去重，保留时间最早的版本
   - 竞品和合作伙伴中排除与盈米新闻重复的内容
4. **排序**：按发布时间倒序排列
5. **生成文档**：创建 Word 格式的舆情周报

## 输出格式

生成的 Word 文档包含：
- 标题：珠海盈米基金销售有限公司舆情监测周报
- 监测结果综述
- 盈米基金重点信息（含媒体、日期、标题、摘要、链接）
- 竞品要闻
- 合作伙伴要闻
- 行业要闻
- 备注说明
- 报告生成时间

## 注意事项

- 确保 Python 环境已安装 `openpyxl` 和 `python-docx` 库
- Excel 文件必须包含指定的工作表名称
- 标题列支持 Excel HYPERLINK 函数格式
- 摘要仅显示包含盈米关键词的内容
- 所有链接保持原格式输出
